# 皇家御苑酒店管理系统

一个基于Spring Boot的现代化Web酒店管理系统，提供完整的酒店运营管理解决方案，包括用户权限管理、房间管理、客户管理、员工管理、预订管理、餐饮订单和停车管理等功能。

## 🏨 系统特色

### 🎨 现代化设计
- **豪华界面**：采用皇家御苑主题，金色配色方案
- **响应式设计**：完美适配桌面和移动设备
- **动态效果**：六个子系统各具特色的交互动画
- **玻璃拟态**：现代化的毛玻璃效果和渐变背景

### 🔐 用户权限管理
- **三级权限**：系统管理员、经理、前台
- **权限控制**：不同角色访问不同功能模块
- **安全登录**：用户身份验证和会话管理
- **角色管理**：灵活的权限分配和管理

## 🚀 功能模块

### 1. 用户登录系统
- **现代化登录界面**：玻璃拟态效果、动画背景
- **权限验证**：基于角色的访问控制
- **会话管理**：安全的用户状态管理
- **多角色支持**：管理员、经理、前台三种角色

### 2. 房间管理
- **房间信息管理**：添加、查看、更新、删除房间
- **房间类型**：单人间、双人间、套房等多种类型
- **状态管理**：可用、已入住、维护中、已预订
- **实时统计**：房间数量、可用率等统计信息

### 3. 客户管理
- **客户档案**：完整的客户信息管理
- **搜索功能**：按姓名、电话等条件搜索
- **客户统计**：客户数量、活跃度等数据分析
- **信息维护**：客户信息的增删改查

### 4. 员工管理（经理权限）
- **员工档案**：员工基本信息管理
- **职位管理**：不同职位的员工分类
- **权限分配**：员工权限和角色管理
- **状态跟踪**：员工在职状态管理

### 5. 预订管理
- **预订流程**：完整的预订创建和管理流程
- **状态跟踪**：已确认、已入住、已退房、已取消
- **自动计算**：预订金额自动计算
- **时间管理**：入住和退房时间管理

### 6. 餐饮订单管理
- **订单处理**：餐饮订单的创建和处理
- **菜品管理**：菜品信息和价格管理
- **订单状态**：待处理、已确认、制作中、已完成
- **支付管理**：订单支付状态跟踪

### 7. 菜品管理
- **菜品信息**：完整的菜品信息管理（名称、描述、价格、分类）
- **分类管理**：热菜、凉菜、汤品、主食、饮品、甜品等分类
- **库存管理**：菜品库存数量跟踪和管理
- **状态控制**：菜品可用/禁用状态管理
- **搜索筛选**：按名称搜索和分类筛选功能
- **CRUD操作**：菜品的增加、查看、编辑、删除操作

### 8. 停车管理
- **停车位管理**：停车位的分配和状态管理
- **车辆记录**：车辆进出记录管理
- **费用计算**：停车费用自动计算
- **实时监控**：停车位使用情况实时显示

### 权限说明

#### 🔴 系统管理员 (ADMIN)
- **权限**：拥有所有功能模块的完全访问权限
- **功能**：用户管理、员工管理、房间管理、客户管理、预订管理、餐饮订单、停车管理
- **特点**：可以创建、编辑、删除所有数据

#### 🟡 经理 (MANAGER)
- **权限**：可以管理员工和所有业务模块
- **功能**：员工管理、房间管理、客户管理、预订管理、餐饮订单、停车管理
- **限制**：不能管理用户账户

#### 🟢 前台 (FRONT_DESK)
- **权限**：只能管理业务模块
- **功能**：房间管理、客户管理、预订管理、餐饮订单、停车管理
- **限制**：不能管理员工和用户

## 🚀 快速开始

### 环境要求
- **Java**: 8 或更高版本
- **Maven**: 3.6 或更高版本
- **内存**: 至少 2GB RAM
- **浏览器**: Chrome、Firefox、Safari、Edge 等现代浏览器

### 安装和运行

1. **克隆项目**：
```bash
git clone <项目地址>
cd hotel_system
```

2. **编译项目**：
```bash
mvn clean compile
```

3. **运行应用**：
```bash
mvn spring-boot:run
```

4. **访问系统**：
   - 🌐 **系统首页**: http://localhost:8080
   - 🔐 **登录页面**: http://localhost:8080/login
   - 🗄️ **数据库控制台**: http://localhost:8080/h2-console

### 数据库配置

系统使用H2内存数据库，配置信息：
- **URL**: `jdbc:h2:mem:hoteldb`
- **用户名**: `sa`
- **密码**: (空)
- **驱动**: `org.h2.Driver`
